O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas da web. Ele estrutura o conteúdo de uma página usando uma série de elementos chamados "tags", que definem o layout e a organização de textos, imagens, links, tabelas, vídeos, entre outros. Cada elemento HTML descreve uma parte específica da página, como títulos, parágrafos, listas, links, ou seções multimídia.
